## AI-Driven Women Safety & Community Intelligence Platform  

HerCodeX is a hybrid **Mobile + Web + AI Intelligence ecosystem** built to provide real-time safety tools, predictive risk awareness, community monitoring, and administrative analytics for women’s safety.

The system is composed of three major layers:

- Mobile Application  
- Web Intelligence Portal  
- AI / ML / DL / LLM Analytics Layer  

---

# Platform Architecture  

HerCodeX operates as a modular three-layer system:

- **Mobile Safety Layer** – User interaction & real-time safety tools  
- **Web Governance Layer** – Monitoring, dashboards & data aggregation  
- **AI Intelligence Layer** – Predictive analytics & anomaly detection  

The mobile layer is lightweight and responsive.  
The intelligence and heavy computation are handled on the web layer.

# AI+ML+DL+LLM Layer  

HerCodeX integrates structured AI components across safety scoring and analytics.

## Safety Scoring Engine  

**Model Type:** Regression Model  

Used for:

- Grid-level safety prediction  
- Police proximity weighting  
- Time-aware safety adjustment  
- Route risk optimization  

Outputs continuous safety score ∈ [0,1].

## NLP Sentiment & Risk Analysis  

**Model Type:** Natural Language Processing (NLP)  

Used for:

- Sentiment classification of community posts  
- Distress detection  
- Weekly emotional trend analysis  
- Risk keyword extraction  

## Anomaly Detection System  

**Model Type:** Isolation Forest  

Used for:

- SOS spike detection  
- Complaint irregularity detection  
- Suspicious behavioral pattern identification  

## LLM Intelligence Layer  

**LLM Used:** Gemini API  

Used for:

- Weekly safety summaries  
- Natural language analytical reports  
- Administrative insight generation  
- Context-aware interpretation of aggregated data  

LLM is used strictly for interpretation and reporting — not routing logic.
